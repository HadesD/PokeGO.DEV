Pokego.dev

Pokemon GO get player inventory and pokemon ivs/information by PHP - Laravel

View project on GitHub

About

Pokemon GO check user info and Pokemon IVs use PHP

  • New pokedex Database
  • Check pokemon 's move
  • more...

Requirements

  • PHP >5.6
  • Composer
  • Git

Install

(cd /server_http_path/)
git clone https://github.com/HadesD/PokeGO.DEV.git
cd /PokeGO.DEV/
composer install
cp .env.example .env
php artisan key:generate

Open browser to view http://localhost/PokeGO.DEV/
